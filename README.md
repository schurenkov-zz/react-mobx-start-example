# react-mobx-start-example

API http://strainapi.evanbusse.com - it's joke!

Demo: https://react-mobx-start-example.herokuapp.com/

Client application on React 16.8

### Installing

#### Install node package

```
npm i
```

or

```
yarn
```

#### Launch of the project

```
npm run start
```

or

```
yarn start
```

#### Build project

```
npm run build
```

or

```
yarn build
```

### Stack

- React (TypeScript)
- React-Router
- MobX
